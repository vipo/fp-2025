queryAge :: String -> IO Integer Source #

Take a list as a traversable, pass a list of Maybies. >>> sequenceA [Just 42, Just 5] Just [42,5] | Returns a Maybe containing a list.

If the passed list contains a Nothing, Nothing is returned. >>> sequenceA [Just 42, Just 5, Nothing] Nothing

Works pretty much as list comprehension: >>> sequenceA [[42], [13, 45]] [[42,13],[42,45]]

Returns empty list: >>> sequenceA [[42], [13, 45], []] []

Because a Left value was passed, it also returns a Left value: >>> sequenceA [Left 0, Right 43, Right 45] Left 0

Returns the inner value (Left 45)

Swaps the traversable with applicative: >>> sequenceA $ Just (Right 45) Right (Just 45)

Collects all the inputs: >>> :t sequenceA [getLine, getLine, getLine] sequenceA [getLine, getLine, getLine] :: IO [String] | Inner values are Just monadic values, type is IO [String]. | Result is an IO list of pure values, which makes life easier.
